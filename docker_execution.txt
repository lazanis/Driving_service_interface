BEFORE:
remove pkg_resources==0.0.0 from requirements.txt

DOCKER:
1. make Dockerfile
2. (sudo) docker image build -t NAME(driving-service-interface-image) .
3. (sudo) docker image ls -> list all images
4. (sudo) docker container ls -> list all containers
# 5. (sudo) docker run -i -p 777:7777 NAME(driving-service-interface-image) -> first port is host, second port is inside container (actual app port)
5. (sudo) docker run -i --network="host" NAME(driving-service-interface-image)
6. (sudo) docker stop CONTAINER_ID
7. (sudo) docker system prune
